Assignment

Write a function "joinNames" that takes two parameters, joins them and returns the joined string.

Example

let firstname = "Max";
let lastname = "Muster";
console.log(joinNames(firstname,lastname)); // "Max Muster"
console.log(joinNames("Herold","Savanah")); // "Herold Savanah"
console.log(joinNames("Anna","Smith")); // "Anna Smith"
