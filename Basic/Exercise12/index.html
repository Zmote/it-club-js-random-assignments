Assignment

Write a function randomizeCharacters, that takes one input string and randomizes its characters. Duplicates are allowed.

Example

    
        let name = "Max Muster";
        console.log(randomizeCharacters(name)); // possible outputs: "xaaM ueMmre", "  xM MsreM"

Possible Solution

    
    function randomizeCharacters(input) {
    let randomizedInput = "";
    for(let i = 0; i < input.length; i++){
        randomizedInput += input[Math.floor(Math.random()*input.length)];
    }
    return randomizedInput;
    }

    console.log(randomizeCharacters("Max Muster"));
    console.log(randomizeCharacters("Max Muster"));
    console.log(randomizeCharacters("Max Muster"));
    console.log(randomizeCharacters("Max Muster"));
