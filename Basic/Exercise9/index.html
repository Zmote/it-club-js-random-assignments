Assignment

Write a function "makeCopy" that keeps adding the string "copy" to the input.

Example

let someInput = "Max";
someInput= makeCopy(someInput); // "Max copy"
someInput= makeCopy(someInput); // "Max copy copy"
someInput= makeCopy(someInput); // "Max copy copy copy"
someInput= makeCopy(someInput); // "Max copy copy copy copy"
console.log(someInput); // "Max copy copy copy copy copy"
