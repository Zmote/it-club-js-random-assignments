Assignment

Write a function "personArray" that takes 5 parameters and returns them as an array.

Example

personArray("Max", "Muster", 25, "Developer", "Switzerland"); // ["Max", "Muster", 25, "Developer", "Switzerland"];
personArray("Anna", "Smith", 22, "Designer", "Scottland"); // ["Anna", "Smith", 22, "Designer", "Scottland"];
