Assignment

Write a function that takes one parameter and returns wether the input is a string, a number, a boolean or something else

Example

let numberVar = 3;
let stringVar = "Max";
let booleanVar = true;
let undefinedVar = undefined;
let nullVar = null;
console.log(typeOf(numberVar)); // "number"
console.log(typeOf(stringVar)); // "string"
console.log(typeOf(booleanVar)); // "boolean"
console.log(typeOf(undefinedVar)); // "other" --> not: "undefined"
console.log(typeOf(nullVar)); // "other" --> not: "object"
