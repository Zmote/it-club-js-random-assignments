Assignment

Write a function that takes a matrix of numbers (in any configuration n x m, n x n, m x n) and randomized it's eyes/values. A number from the source matrix can only be used once in the new matrix.

Example

let matrix = [[1,2],[3,4]];
console.log(randomizeMatrix(matrix)); // possible outputs: --> [[2,3],[1,4]], [[1,3],[2,4]]
